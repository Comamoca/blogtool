## 🚀 How to use

First, set `BLOG_PATH` Enviroment Valiable to your shell.
`BLOG_PATH` is path to markdown directory on your blog.

```sh
# show blog content path same to `echo $BLOG_PATH`.
blog path

# create new articles interactively.
blog new post
```
